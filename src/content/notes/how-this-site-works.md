This section is a garden, not a blog. Notes land here when a conversation or a
build produces something worth keeping. There's no theme and no schedule.

## How notes get here

Each note is a markdown file in a git repository. Pushing to the main branch
rebuilds and redeploys the whole site automatically. That's the entire
publishing pipeline — no CMS, no editor, no login.

Notes can reference each other with wiki-style links, like this one pointing
back to [[how-this-site-works|itself]]. Over time the links matter more than
the notes: clusters form, and a cluster that gets dense enough might break off
into its own site.

## Why bother

Writing things down is cheap. Losing them is expensive. This is the place
where they don't get lost.
